# Karel, virtuální asistent
Karel je asistent běžící na LLM modelu Gemma 2 od googlu.  

## Instalace
Pro funkčnost Karla je nutné [doinstalovat Ollamu](https://ollama.com/download) a Python knihovny přes PIP:
```
pip install -r requirements.txt
```
## Spouštění
Pro větší jednodost je v souborech přidán .bat soubor, který slouží k rychlému spouštění programu.

## Verdikt
Cílem celé práce bylo zhotovit lokálního AI Asistenta, který nahrazuje korporátní modely (např. Anthropic, Google, OpenAI...)

### Výhody
Model běží kompletně lokálně a offline, takže je 100% soukromý a důvěryhodný. Narozdíl od veřejných modelů, kde společnosti sbírají veškeré data uživatelů.

### Nevýhody
I se základním hardwarem je model extrémně pomalý a nepraktický. Občas může poskytovat nesmyslný výstup.
