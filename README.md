# Karel, virtuální asistent
Karel je asistent běžící na LLM modelu Gemma 2 od googlu.  

## Instalace
Pro funkčnost Karla je nutné [doinstalovat Ollamu](https://ollama.com/download) a Python knihovny přes PIP:
```
pip install -r requirements.txt
```
# Spouštění
Pro větší jednodost je v souborech přidán .bat soubor, který slouží k rychlému spouštění programu.
